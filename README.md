# PillSmart – Smart Healthcare & Medicine Dispensing System

PillSmart is a healthcare automation solution developed during Smart India Hackathon (SIH), designed to digitize prescriptions and enable secure, automated medicine dispensing.

---

## Problem

Traditional healthcare systems face several issues:

- Long queues at pharmacies  
- Limited accessibility (time & location constraints)  
- Poor medication tracking  
- Fragmented patient medical records  

---

## Solution

PillSmart introduces a digital-first healthcare system combining a web platform with automated medicine dispensing.

### Key Features

- Digital prescription management  
- Patient, doctor, and admin dashboards  
- OTP-based secure medicine dispensing  
- Centralized patient medical records  
- Real-time medicine inventory tracking  

---

## System Workflow

Patient → Doctor → Prescription Upload → System → Vending Machine → OTP Verification → Medicine Dispensed

---
